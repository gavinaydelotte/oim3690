Learning Logs

Weekly reflections from OIM3690 — Web Development.

Week 4 — HTML Basics

Learned to write HTML and was introduced to semantic elements like <header>, <p>, <ol>, <ul>, and <mark>. Learned how to embed images and audio, and got an introduction to CSS. Also started planning my first client project — a website for my mom's vacation rental business.

Week 5 — CSS Styling

Learned more intricate CSS, including how to change colors and fonts, and how cascading works — the last rule overrides earlier ones. Began building the basic outline for my mother's vacation rental website, modeled after Airbnb and VRBO.

Week 6 — Selectors, Flexbox & Grid

Completed CSS Diner and learned:

  • CSS selectors (there are a lot of them)
  • Flexbox layout
  • CSS Grid layout
  • Hover and transition effects

Week 7 — Forms & Responsive Design

Covered HTML forms and CSS frameworks. Learned <form>, <input>, <label>, <textarea>, and input types like text, email, and password. Also covered responsive design with the viewport meta tag and an introduction to Tailwind CSS.

Week 8 — JavaScript Intro

Started learning JavaScript. Added audio and video elements to a web page, implemented collapsing text, and reviewed CSS frameworks like Tailwind and Bootstrap. Started to see addEventListener everywhere — still working out exactly how it fits together.

Week 9 — let, const & Fetch

Built a JavaScript Todo app (Mini-Project 2 — a health and fitness tracker). Learned let vs. const: use const by default, switch to let only when the value needs to change. Also covered numbers and math in JavaScript using the Math object.

Week 10 — Conditionals, Loops & Arrays

Learned JavaScript fundamentals including if/else conditionals, strict vs. loose equality (=== vs. ==), and logical operators. Also covered how loops and arrays work together using for and while. Finished Mini-Project 2 and began preparing for the JavaScript quiz.

Week 11 — APIs, JSON & Async

Wrapped up loops and covered APIs, JSON, and the fetch function. Learned that async/await is required when fetching data from external sources.

Week 12 — API Keys & Mini-Project 3

Learned about API keys and how to protect them using .gitignore. Used an OpenAI key to build a textbox that responds to questions using AI. Also took the JavaScript quiz and worked on Mini-Project 3, PokeMatchup.

Week 13 — Timers, Storage & Deployment

Covered extra JavaScript features including setTimeout, setInterval, localStorage, and the Geolocation API. Got an overview of frameworks like React, Vue, Next.js, and Svelte. Discussed deployment options — GitHub Pages, Vercel, and Render — and began preparing for Demo Day. Started my final project: a quiz creator using AI to generate flashcards, with a multiplayer mode using Firebase.
