## Week 11 Learning Log

This week, we wrapped up learning about loops and combining them with scripts. We also covered several new topics:

- **APIs** — the way programs talk to each other; provides external data to your code that can be fetched using keys and linking with a server
- **JSON** — the text format for data
- **`fetch`** — used to retrieve API data and convert it to JSON
- **`async` / `await`** — mandatory when fetching data
