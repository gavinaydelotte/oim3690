# Week 7 Learning Log

## What I Learned

This week focused on **HTML forms** and **CSS frameworks** — the building blocks for collecting user input on the web.

### HTML Forms
- `<form>`, `<input>`, `<label>`, `<textarea>`, and `<button>` elements
- The `action` and `method` attributes — how forms send data
- Input types: `text`, `email`, `password`, `submit`
- Building contact forms and login/signup forms

### Responsive Design & CSS Frameworks
- Mobile scaling with the `viewport` meta tag
- Introduction to CSS frameworks (like Tailwind CSS)
- How frameworks speed up styling with utility classes

## Key Takeaway

Forms are how websites collect information from users — whether it's a login, a search bar, or a contact form. Getting the structure and input types right matters for both usability and accessibility.
